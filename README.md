# Engineering Interactive Exam
An interactive engineering exam platform combining MCQ questions, short essays, and technical reports. It's designed to help engineering students review their technical report writing and engineering English curriculum in a random and interactive way.
## ✨ Key Features
- 15 randomly selected questions from a pool of 94 real questions (no duplication in any exam).
- Filter by topic (Grammar, CV, Motivation, Recommendation, Intro, Clauses, Comprehension, Technical...).
- Filter by question type (MCQ – Essay – Report).
- Instant MCQ evaluation with correct and incorrect answers displayed, and answers highlighted (green/red) in day and night mode.
- A writing area for essay and report answers with the option to view a sample answer and self-assess using keywords.
- Day/Night mode (Dark/Light mode).
- Automatic saving of answers to local storage.
- A results screen displaying the percentage, errors, and their analysis.
- Full keyboard support (Tab/Shift+Tab/Enter/Space).
## 🛠️ Technologies Used
- HTML5 + CSS3 (Flex/Grid responsive design).
- JavaScript (ES6).
- JSON (question database).
- GitHub Pages for publishing.

## 🚀 How to Run
### Locally (Offline)
1. Download the index.html and questions.json files into the same folder.
2. Open index.html using any modern browser (using a live server is recommended to avoid CORS issues).
### Via GitHub Pages
1. Upload the two files to a public repository on GitHub.
2. Enable GitHub Pages from Settings (Settings → Pages → Branch: main).
3. Open the link below.
